== Description ==

This plugin connects to your site the opportunity to comment VKontakte users



== Installation ==

How to install the plugin and get it working.


* 1) Copy folder with this plugin to wp-content/plugins directory
* 2) Activate plugin
* 3) Set options on settings page (Comments -> VK Comments)

== Frequently Asked Questions ==

[English] What is the option VKontakte API ID 
Perform the login to your Vkontakte account. Go to the "Connecting the comments on your site" page http://vkontakte.ru/developers.php?o=-1&p=Comments and copy your number of VKontakte API ID, for example: "apiId: 1960175"
                
[Russian] Что такое опция VKontakte API ID
Выполните логин в ваш аккаунт Vkontakte. Перейдите на страницу "Подключение комментариев к Вашему сайту". http://vkontakte.ru/developers.php?o=-1&p=Comments В поле "Код для вставки:" найдите и скопируйте цифровую часть VKontakte API ID, для примера: "apiId: 1960175"
